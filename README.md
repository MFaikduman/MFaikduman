## Selected Projects

### HOMTECH: Emotion Analysis & Smart Home System
Real-time facial emotion recognition that triggers automated smart home scenarios. Built with deep learning to detect emotions and convert them into actionable home automation events.
- **Stack**: Python, TensorFlow, Keras, OpenCV, Flask
- **Key Features**: Custom CNN model, real-time emotion detection, scenario automation
- **Status**: Active development

### Simurg Rocket Mission Dashboard
Mission tracking and test data management platform for TEKNOFEST rocket projects. Features complete CRUD operations, filtering, and persistent data storage with a responsive interface.
- **Stack**: React, Vite, JavaScript, Tailwind CSS
- **Key Features**: Real-time filtering, localStorage persistence, collaborative workflow
- **Status**: Production

### House Price Prediction Model
End-to-end machine learning project for real estate price estimation. Includes comprehensive data exploration, feature engineering, and model comparison.
- **Stack**: Python, Pandas, NumPy, scikit-learn
- **Key Features**: Data preprocessing, Lasso Regression, Random Forest comparison, performance evaluation
- **Status**: Complete

### TÜBİTAK VR Training Simulations
Immersive VR environment for occupational safety and healthcare education. Interactive scenarios with realistic physics and performance tracking.
- **Stack**: Unity, C#, VR integration, 3D modeling
- **Key Features**: Scenario-based learning, physics interaction, safety compliance training
- **Status**: Active development

### Baboon Crossing Synchronization Problem
Operating systems project demonstrating concurrent resource management using POSIX threads, mutex locks, and semaphore-based synchronization.
- **Stack**: C++, POSIX, System Programming
- **Key Features**: Thread synchronization, deadlock prevention, classical OS problem solving
- **Status**: Complete
